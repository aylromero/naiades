Similar a la integración que existe entre ClearPass y otros proveedores, Check Point admite en un nivel básico la capacidad de pasar los atributos de nombre de usuario y dirección IP de origen. Pero también se pueden pasar otros atributos que se muestran a continuación. Como resumen, esta es una lista de los atributos que se pueden pasar de ClearPass a Check Point que se han probado en nuestro laboratorio.
Atributos Check Point
IP de origen
Nombre de usuario
Rol del usuario
Dominio
Tipo de dispositivo [b]
MachineOS [b]
Nombre de la máquina [a]
Heath / Postura
[a] = Disponible desde llamadas API REST HTTP no desde RADIUS Accounting.
[b] = Requiere el punto de control R80.XX o R77.XX con HOTFIX
